# Episode Packet Contract

Every topic folder should contain these files in this order:

## `00_topic-angle.md`

- core question
- click reason
- emotional reason
- viewer starting assumption
- changed viewer state at the end

## `01_thumbnail-hook.md`

- thumbnail text candidates
- first-image promise
- opening anomaly or anecdote
- first-30-seconds proof

## `02_evidence-map.md`

- strongest case
- supporting cases
- facts that must stay exact
- weak claims to avoid
- modern analogy options
- for structure-heavy topics: actors, mechanism, response, human cost, interested-party claims, unresolved choice

## `03_outline.md`

- opening
- example cluster
- middle turn 1
- middle turn 2
- modern re-entry
- ending reinterpretation

## `04_segment-map.md`

- segment goal
- tension question
- proof used
- transition line
- payoff type
- host reaction line when useful

## `05_script-draft.md`

- full spoken draft
- may contain revision notes

## `06_final-script.md`

- narration only
- no planning labels
- no evidence tables
- no production notes

## `07_publish-card.md`

- packaging summary
- callback to older episode
- sequel seed
- retention risk guess
- postmortem notes
