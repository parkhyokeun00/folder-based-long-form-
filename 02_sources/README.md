# Sources Layer

This folder is the evidence intake layer between browsing and narration.

- `raw/`: untouched capture
- `research/`: ratified research notes for contested or source-sensitive topics
- `parsed/`: normalized extracts, case summaries, and reusable proof
- `canon-integrated/`: claims stable enough to reuse across episodes
- `deprecated/`: rejected, outdated, or misleading material

Read `00_system/SOURCE_INGEST_PROTOCOL.md` before promoting any source into script truth.
