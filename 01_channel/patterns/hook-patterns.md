# Hook Patterns

This page stores reusable opening patterns that should survive beyond any single topic.

## Pattern 1. Anomaly first

- start with the strangest concrete scene, not the thesis
- make the viewer feel the mismatch before naming it
- good default move: familiar scene -> hidden contradiction -> `그런데`

## Pattern 2. Example cluster

- when the topic is cultural, behavioral, or rhetorical, do not rely on one polite sample
- front-load multiple fast examples so the pattern becomes obvious before the explanation
- default target: at least four quick examples before the first long explanation block

## Pattern 3. First-minute proof

- the first minute must prove the thumbnail promise, not merely restate it
- if the click promise is `친절한데 무섭다`, the first minute should already show the kindness and the threat in the same breath

## Pattern 4. Host-side reaction

- the opening should include at least one host sentence that reacts to the material
- examples:
- `이쯤 되면 느낌이 옵니다`
- `친절한데 서늘하죠`
- `여기서 핵심은 따로 있습니다`

## Pattern 5. Household choice versus system shock

- for luxury, commodity, energy, and supply-chain topics, start with a personal or household choice
- then reveal the larger system, state, or labor shock created by that same choice
- good default move: private purchase -> hidden bill -> distant consequence
