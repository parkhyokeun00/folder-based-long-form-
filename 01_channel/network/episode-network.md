# Episode Network

This page stores truthful bridges between topics so the channel can feel like a growing knowledge world rather than isolated uploads.

## Network rules

- callbacks must add meaning, not just self-reference
- sequel seeds should feel inevitable after the current ending, not bolted on
- when no truthful bridge exists, do not force one

## Expansion method

For each new finished topic, record:

- what older topic it clarifies
- what next topic it naturally opens
- what common mechanism it shares with the rest of the channel
